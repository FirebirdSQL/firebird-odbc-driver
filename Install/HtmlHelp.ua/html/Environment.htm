Середовище

Firebird ODBC драйвер дозволяє одночасно виконувати з'єднання з різноманітними базами даних і різноманітними серверами. Кожне з'єднання працює в своєму спецефічному середовищі. Іншими словами, якщо прикладна задача використовує одночасно два з'єднання Firebird server і Firebird embeded, то драйвер для сервера Firebird загрузить клиєнта fbclient.dll(libfbclient.so) і для вбудованного сервера FbEmbeded загрузить fbembed.dll(libfbembed.so).

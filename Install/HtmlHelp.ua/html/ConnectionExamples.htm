Приклади З'єднання

Приклади строк з'єднання для Прикладної задачі яка використовує ODBC API функцію SQLDriverConnect:

1. Open("DSN=myDb;")

2. Open("DSN=myDb; UID=MCSSITE; PWD=mcssite;")

3. Open("DSN=myDb; UID=MCSSITE; PWD=mcssite; DBNAME=172.17.2.10:/usr/local/db/myDb.fdb;")

4. Open("DRIVER=Firebird/InterBase(r) driver; DBNAME=172.17.2.10:/usr/local/db/myDb.fdb;")

5. Open("DRIVER=Firebird/InterBase(r) driver; UID=MCSSITE; PWD=mcssite; DBNAME=172.17.2.10:/usr/local/db/myDb.fdb;")

також

6. Open("DRIVER=Firebird/InterBase(r) driver; UID=MCSSITE; PWD=mcssite; DBNAME=dummy;")

dummy – є псевдонімом, отриманим з Firebird aliases.conf файла. Якщо змінні середовища ISC_PASSWORD і ISC_USER встановлені, то їх значення також використовуються.

DBNAME строка для віддаленного з'єднання Коментарі
172.17.2.10:/usr/local/db/myDb.fdb використання IP адреси сервера, з іменем файла в стилі Unix запису
myserver:/usr/local/db/myDb.fdb використання імені сервера, з іменем файла в стилі Unix запису
172.17.2.10/3051:/usr/local/db/myDb.fdb використання IP адреси сервера і альтернативний порт, з іменем файла в стилі Unix запису
myserver/3051:/usr/local/db/myDb.fdb використання імені сервера і альтернативний порт, з іменем файла в стилі Unix запису
172.17.2.10:c:\db\myDb.fdb використання IP адреси сервера, з іменем файла в стилі Windows запису
myserver:c:\db\myDb.fdb використання імені сервера, з іменем файла в стилі Windows запису
172.17.2.10/3051:c:\db\myDb.fdb використання IP адреси сервера і альтернативний порт, з іменем файла в стилі Windows запису
myserver/3051:c:\db\myDb.fdb використання IP адреси сервера і альтернативний порт, з іменем файла в стилі Windows запису
127.0.0.1:/usr/local/db/myDb.fdb використання петльового інтерфейса, з іменем файла в стилі Unix запису
localhost:/usr/local/db/myDb.fdb використання петльового інтерфейса, з іменем файла в стилі Unix запису
127.0.0.1:c:\db\myDb.fdb використання петльового інтерфейса, з іменем файла в стилі Windows запису
localhost:c:\db\myDb.fdb використання петльового інтерфейса, з іменем файла в стилі Windows запису

DBNAME строка для місцевого з'єднання Коментарі
C:\db\myDb.fdb місцеве з'єднання, з іменем файла в стилі Windows запису
/usr/local/db/myDb.fdb місцеве з'єднання, з іменем файла в стилі Unix запису

Приклад з використанням псевдоніма: Використання Firebird aliases.conf - файл псевдонімів.

    dummy = c:\data\myDb.fdb
    або
    dummy = /usr/local/db/myDb.fdb

DBNAME строка з використанням псевдонімів Коментарі
172.17.2.10:dummy віддалене з'єднання з використанням IP адреси сервера і псевдоніма
myserver:dummy віддалене з'єднання з використанням імені адреси сервера і псевдоніма

Примітка. При використанні псевдонімів для клиєнта не має значення який використовується сервер - Unix чи Windows, Ви вказуєте тільки ім'я псевдоніма. Фактичне місцезнаходження бази визначено в файлі псевдонімів aliases.conf.

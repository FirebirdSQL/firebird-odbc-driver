Коротко про Firebird ODBC драйвер

Firebird ODBC драйвер підтримує Firebird для Windows, FreeBSD, Solaris, і Linux.

Firebird ODBC драйвер складається з слідуючих файлів:

OdbcFb.dll

Забеспечує ODBC API інтерфейс, є провайдером для ODBC Менеджера. В прикладних задачах може використовуватися без ODBC Менеджера. Має додатково інтерфейс JDBC, забезпечує доступ до Firebird через Firebird API. Також виконує роль інсталятора (деінсталятора) Firebird ODBC драйвера і конфігуровача DSN.

OdbcJdbc.chm

Допомога по використанню Firebird ODBC драйвера.

Firebird ODBC driver:

    - на основі Ms SDK можливо зібрати Win64.

    - підтримка Unicode.

    - потоко-безпечна обробка запитів.

    - можливе створення бази даних за допомогою функцій SQLConfigDataSource, SQLDriverConnect, SQLExecDirect.

    - на одному з'єднанні можливо виконувати одночасно більше однієї транзакції. Транзакції по суті можуть бути різними. Одна "читаюча", інша "пишуча", і т.д.

    - дозволяється створення пула підготовлених описів транзакцій, з присвоєнням їм імен і робочого простору. Для подальшого використання без виконання підготовки(prepare).

    - підтримується використання механізму подій Firebird сервера.

    - підтримується використання Microsoft ODBC cursors( odbccr32.dll, odbccu32.dll ).

    - можливе використання сервісу (backup, restore, statistics, repair) бази даних за допомогою функції SQLConfigDataSource.

    - для сумісності SQL-запитів підтримується використання схем(SCHEMA or OWNER).

    - підтримується повно-функціональний SQL синтаксис менеджера транзакцій(Firebird GPRE).

    - підтримується використання COM технології Microsoft DTC.

Коротко про Firebird ODBC драйвер

Firebird ODBC драйвер підтримує Firebird для Windows, FreeBSD, Solaris, і Linux.

Firebird ODBC драйвер складається з слідуючих файлів:

IscDbc.dll

Інтерфейс JDBC, забезпечує доступ до Firebird через Firebird API.

OdbcJdbc.dll

Забеспечує ODBC API інтерфейс, є провайдером для ODBC Менеджера. В прикладних задачах може використовуватися без ODBC Менеджера.

OdbcJdbcSetup.dll

Виконує роль інсталятора (деінсталятора) Firebird ODBC драйвера і конфігуровача DSN.

OdbcJdbc.chm

Допомога по використанню Firebird ODBC драйвера.

Использование Firebird ODBC драйвер с Clarion

Clarion пользователи могут работать со смешанными именами объектов в Firebird, при условиях указанных ниже.

  1. Создайте вашу базу данных в Firebird. Вы можете использовать названия таблиц подобно "Pending_Invoices" и полям подобно "Order_Number".
  2. Создайте DSN для базы данных, и объязательно отметьте все опции в разделе "Расширенные Реквизиты Идентификатора"
  3. Откройте ваш словарь, и импортируйте множественные таблицы из odbc источника. Это будет работать, но не пробуйте просматривать или использовать файлы в приложении.
  4. Для каждого поля, укажите 'Внешнее Имя' имя поля, окруженного кавычками (например, укажите "Order_Number" во внешнем имени).
  5. Это все что нужно было сделать! Теперь используйте ваш словарь с Mixed_Case_Identificators без проблем. Но помните - Вы должны использовать двойные кавычки вокруг объектных названий во всех SQL утверждениях внутри Clarion.

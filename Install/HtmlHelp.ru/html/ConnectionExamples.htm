Примеры соединения

Примеры строки соединения для Приложения использующего ODBC API функцию SQLDriverConnect:

1. Open("DSN=myDb;")

2. Open("DSN=myDb; UID=MCSSITE; PWD=mcssite;")

3. Open("DSN=myDb; UID=MCSSITE; PWD=mcssite; DBNAME=172.17.2.10:/usr/local/db/myDb.fdb;")

4. Open("DRIVER=Firebird/InterBase(r) driver; DBNAME=172.17.2.10:/usr/local/db/myDb.fdb;")

5. Open("DRIVER=Firebird/InterBase(r) driver; UID=MCSSITE; PWD=mcssite; DBNAME=172.17.2.10:/usr/local/db/myDb.fdb;")

также

6. Open("DRIVER=Firebird/InterBase(r) driver; UID=MCSSITE; PWD=mcssite; DBNAME=dummy;")

dummy – является псевдонимом, полученным из Firebird aliases.conf файла. Если переменные среды ISC_PASSWORD и ISC_USER установлены, то их значение также используется.

DBNAME строка для удаленного соединения Коментарий
172.17.2.10:/usr/local/db/myDb.fdb использование IP адрес сервера, с именем файла в стиле Unix записи
myserver:/usr/local/db/myDb.fdb использование имени сервера, с именем файла в стиле Unix записи
172.17.2.10/3051:/usr/local/db/myDb.fdb использование IP адрес сервера и альтернативный порт, с именем файла в стиле Unix записи
myserver/3051:/usr/local/db/myDb.fdb использование имени сервера и альтернативный порт, с именем файла в стиле Unix записи
172.17.2.10:c:\db\myDb.fdb использование IP адрес сервера, с именем файла в стиле Windows записи
myserver:c:\db\myDb.fdb использование имени сервера, с именем файла в стиле Windows записи
172.17.2.10/3051:c:\db\myDb.fdb использование IP адрес сервера и альтернативный порт, с именем файла в стиле Windows записи
myserver/3051:c:\db\myDb.fdb использование IP адрес сервера и альтернативный порт, с именем файла в стиле Windows записи
127.0.0.1:/usr/local/db/myDb.fdb использование петлевого интерфейса, с именем файла в стиле Unix записи
localhost:/usr/local/db/myDb.fdb использование петлевого интерфейса, с именем файла в стиле Unix записи
127.0.0.1:c:\db\myDb.fdb использование петлевого интерфейса, с именем файла в стиле Windows записи
localhost:c:\db\myDb.fdb использование петлевого интерфейса, с именем файла в стиле Windows записи

DBNAME строка для локального соединения Коментарий
C:\db\myDb.fdb локальное соединение, с именем файла в стиле Windows записи
/usr/local/db/myDb.fdb локальное соединение, с именем файла в стиле Unix записи

Пример с использованием псевдонима: Использование Firebird aliases.conf - файл псевдонимов.

    dummy = c:\data\myDb.fdb
    or
    dummy = /usr/local/db/myDb.fdb

DBNAME строка с использованием псевдонимов Коментарий
172.17.2.10:dummy удаленное соединение с использованием IP адреса сервера и псевдонима
myserver:dummy удаленное соединение с использованием имени адреса сервера и псевдонима

Примечание. При использовании псевдонимов для клиента не имеет значение какой используется сервер - Unix или Windows, Вы определяете только имя псевдонима. Реальное расположение базы определено в файле псевдонимов aliases.conf.

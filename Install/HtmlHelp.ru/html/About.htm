Кратко про Firebird ODBC драйвер

Firebird ODBC driver поддерживает Firebird для Windows, FreeBSD, Solaris, и Linux.

Firebird ODBC драйвер состоит из следующих файлов:

OdbcFb.dll

Обеспечивает ODBC API интерфейс, является провайдером для ODBC Менеджера. В прикладных задачах может использоваться без ODBC Менеджера.

Имеет дополнительно интерфейс JDBC, обеспечивая доступ к Firebird через Firebird API.

Также выполняет роль инсталлятора (деинсталлятора) Firebird ODBC драйвера и конфигуратора DSN.

OdbcFb.chm

Помощь по использованию Firebird ODBC драйвера.

Firebird ODBC driver:

    - на базе Ms SDK возможна сборка Win64.

    - поддержка Unicode.

    - потоко-безопасная обработка запросов.

    - возможно создание базы данных посредством функций SQLConfigDataSource, SQLDriverConnect, SQLExecDirect.

    - на одном соединении возможно выполнять одновременно больше одной транзакции. Транзакции по сути могут быть разными. Одна "читающая", другая "пишущая", и т.д.

    - допускается создание пула подготовленных описаний транзакций, с присвоением им имен и рабочего пространства. Для последующего использования без выполнения подготовки(prepare).

    - поддерживается использование механизма событий Firebird сервера.

    - поддерживается использование Microsoft ODBC cursors( odbccr32.dll, odbccu32.dll ).

    - возможно использование сервиса (backup, restore, statistics, repair) базы данных посредством функции SQLConfigDataSource.

    - для совместимости SQL-запросов поддерживается использование схемы(SCHEMA or OWNER).

    - поддерживается полнофункциональный SQL синтаксис менеджера транзакций(Firebird GPRE).

    - поддерживается использование COM технологии Microsoft DTC.
